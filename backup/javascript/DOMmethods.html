DOM Operations

Fetching Element
document.getElementsByTagName('section') //getting elements by tag
document.getElementsByClassName('list-group') // getting elements by class
document.getElementById('sidebar') // getting element by id
document.getElementsByName('myField') //getting elemenets by name
ID selector returns just one element but other selectors returns an array of elements. Therefore, you should use [0] with them if you have only 1 element. If you need more than one element then iterate through them.
Iterating through class array
var nodelist = document.getElementsByClassName('list-group-items');
var listLength = nodelist.length;

for(var i=0;i<listLength;i++){ 
	// Use nodelist[i]
}
Query selector
document.querySelectorAll(".myclass");
document.querySelectorAll("#myID"); 
//iterating through classlist using query selector and foreach

document.querySelectorAll('.edit').forEach(function(button) {
    // Now do something with my button
});
Query Selectors provides a jquery like simple syntax to select the elements. It is supported in all major browsers.

Adding and Removing Classes

Classlist property has 4 methods -

  • add - adds a class
  • remove - removes a class
  • toggle - toggles a class
  • contains - checks if a class exists
Classlist
// adds class "classname" to el
el.classList.add("classname");

// removes class "classname" from el
el.classList.remove("classname");

// toggles the class "foo"
el.classList.toggle("foo");

// outputs "true" to console if el contains "foo", "false" if not
console.log( el.classList.contains("foo") );

// add multiple classes to el
el.classList.add( "foo", "classname" );
