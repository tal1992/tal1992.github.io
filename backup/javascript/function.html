Functions in Javascript

Functions are a nothing but a block of code which needs to be called in order to be executed.

Advantages of using Functions are :

  • It reduces the redundancy of code at different points in the program.
  • It makes the program modular and easy to understand.
  • It saves time and efforts of the developer.
  • It makes debugging easier as errors have to be corrected once and at one point only.
  • It avoids repetition of code

Calling Function

Invoking Function
function sum(x,y){
    return x*y;
}

var result = sum(10,20);
console.log(result); //200
